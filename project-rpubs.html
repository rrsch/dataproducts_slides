Core Algorithm

x_std_dev <- 1.0
y_std_dev <- 2.0
obs<-100
x <- rnorm(obs, sd=x_std_dev)
y <- rnorm(obs, sd=y_std_dev)
dat<-data.frame(x,y)
lm(dat$y~dat$x)


Call:
lm(formula = dat$y ~ dat$x)

Coefficients:
(Intercept)        dat$x  
    -0.1493       0.2001
